# 本地 AI 代理服务器

## 快速启动（3步）

### 1. 填入 API Key
打开 `local-proxy/.env` 文件，把 `sk-在这里填入你的API Key` 替换成你的阿里云百练 API Key。

### 2. 启动服务器
```
node local-proxy/proxy-server.js
```

看到以下输出说明成功：
```
  ┌─────────────────────────────────────────────┐
  │  AI 代理服务器已启动                          │
  │  地址: http://localhost:3000                  │
  │  API Key: 已配置 ✓                            │
  └─────────────────────────────────────────────┘
```

### 3. 在素材库网页里设置
打开素材库 → 点右上角「AI 设置」→ 填入：
```
http://localhost:3000
```

点保存，AI 功能就能用了。

---

## 注意事项

- 服务器运行期间 AI 功能可用，关闭终端后会停止
- 每次使用前需要重新运行 `node local-proxy/proxy-server.js`
- API Key 只存在本地 `.env` 文件中，不会上传到任何地方
- 需要保持终端窗口开着，关了就停了
